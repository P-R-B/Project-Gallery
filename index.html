2. Project Gallery: This Page

This page is a collection of all the projects I am working on as part of my application to Founders and Coders. The sections are arranged using the Flexible Box Layout.

Responsive Design

On large-screen devices (e.g. PC monitors), the first four sections are arranged in a configuration of two rows by two columns, with the fifth column directly underneath.

On small screen devices (e.g. smartphone displays), all five sections are arranged in a single column running down the page.

Standardisation

All projects share a base visual format, using the same Arial font and box structure.
The main variance is with the colour scheme used for each project.

Not only does this approach give my projects some consistency, however it also simplifies the coding, as the CSS and JavaScript could be replicated and reused amongst the projects.
